# Project Rules for Claude

This is a mobile web game.

Before opening a PR:
- Understand the existing code structure.
- Make the smallest safe change.
- Do not rewrite the whole app unless asked.
- Run npm install if needed.
- Run npm run build before finishing.
- Create a PR with a clear summary.
- Link the PR to the issue using "Fixes #issue-number".

For gameplay changes:
- Keep controls mobile-first.
- Avoid adding heavy dependencies.
- Keep performance smooth on mobile.
